===============================================
🖌️ APLIKASI MENGGAMBAR SEDERHANA - PYTHON
===============================================

📌 Deskripsi:
Aplikasi ini dibuat untuk tugas mata kuliah Grafika Komputer. 
Dengan menggunakan mouse, Anda dapat menggambar:

✔️ Titik  
✔️ Garis  
✔️ Persegi  
✔️ Lingkaran  
✔️ Elips  

Aplikasi mendukung pemilihan warna dan mode gambar melalui antarmuka menu.

---

📂 File:
- main.py — File utama program
- README.txt — File ini

---

⚙️ Cara Menjalankan:

1. Pastikan Python 3.x sudah terpasang di komputer Anda.
2. Install pustaka pygame (jika belum):
   > pip install pygame
3. Jalankan program:
   > python main.py

---

📝 Catatan:
- Tidak diperlukan penyimpanan gambar.
- Gambar akan hilang saat aplikasi ditutup.
- Cocok untuk latihan dasar penggunaan mouse & bentuk-bentuk 2D dalam grafika komputer.
