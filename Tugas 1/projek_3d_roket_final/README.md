# Roket 3D Interaktif

🚀 Proyek ini memvisualisasikan sebuah roket 3D yang dibangun dari bentuk-bentuk dasar 3D menggunakan Python, Pyodide, dan Plotly.

## 📦 Komponen Objek
- **Badan Roket**: Silinder
- **Kepala Roket**: Kerucut
- **Sayap**: 3 buah balok
- **Ekor**: Silinder kecil

## 🔧 Teknologi
- Python (via Pyodide)
- Plotly 3D
- HTML + JavaScript

## ▶️ Cara Menjalankan
1. Upload seluruh isi folder ini ke itch.io (HTML project).
2. Buka `index.html` di browser (jika lokal, gunakan Live Server).

## 📁 Struktur Folder
```
projek_3d_roket/
├── index.html
├── main.py
└── README.md
```
